# 🛒 Projeto: Lista de Produtos (React)

Aplicação desenvolvida com React que permite visualizar, adicionar e persistir produtos utilizando **Local Storage**.

---

## 🚀 Funcionalidades

* 📦 Listagem de produtos em formato de cards
* ➕ Adição de novos produtos via formulário
* 💾 Persistência de dados com Local Storage
* 🔄 Carregamento inicial simulando API (`useEffect + setTimeout`)
* 🖱️ Scroll horizontal estilo carrossel com drag
* ⏳ Tela de carregamento ("Aguarde, carregando itens...")

---

## 🧠 Conceitos aplicados

* Hooks do React:

  * `useState`
  * `useEffect`
  * `useRef`
* Manipulação de arrays (`map`, spread `...`)
* Gerenciamento de estado
* Persistência com `localStorage`
* Eventos de mouse (drag para scroll)
* Condicional de renderização
* CSS responsivo e organização de layout
